CICS Transaction Server for OS/2 Warp, V4.1 (P1)

This product is located on Premier Disc 1.

CICS® Transaction Server for OS/2 Warp confirms the thrust of a simple to install, use and operate philosophy that has been a hallmark of Version 4.0 of this Transaction Server. It provides a comprehensive CICS API but is very sparing in its use of resources, thus performing extremely well in relatively small server machines.

CICS Transaction Server for OS/2 Warp, Version 4.1 contains CICS for OS/2 Version 3.1 and CICS Clients Version 2.0.2, delivering a range of technical enhancements over previous levels.

A new and flexible pricing structure is introduced to CICS Transaction Processing on the OS/2 platform, based on servers and registered users (with no option of concurrent users) at a much reduced price per user.

CICS Transaction Server for OS/2 Warp, Version 4.1 also includes Lotus® Domino™ Go Webserver, for creating and accessing Internet or Intranet-enabled applications on CICS for OS/2.

What's New

  • CEDA dynamic resource installation
  • Support for Multi-Region Operation (MRO)
  • Telnet TN3270 support
  • Front End Programming Interface (FEPI) LU0 support
  • Enhanced Performance Analyzer function

Hardware and Software Requirements

  • Any system running OS/2 Warp Version 3 or later, with memory requirement starting from 16 MB (according to use).

CICS for OS/2 Version 3.1 requires one of the following operating systems:

  • OS/2 Warp Server Version 4
  • OS/2 Warp Version 4
  • OS/2 Warp Connect Version 3
  • OS/2 Warp Version 3
IBM OS/2 for Symmetrical Multiprocessing Version 2.11

The sample command files included in CICS for OS/2 Version 3.1 are written in REXX, therefore, if they are used as supplied, the classic REXX option of OS/2 should be installed.

When using NetBIOS communications in a LAN environment, CICS for OS/2 Version 3.1 requires the presence of an appropriate product that provides the API of NetBIOS 3.0 or later, such as:

  • IBM OS/2 Warp Server Version 4, OS/2 Warp Version 4, or OS/2 Warp Connect Version 3
  • IBM OS/2 LAN Server Version 4.0

NetBIOS communication is also supported over IPX/SPX using the NetBIOS emulator in the Novell NetWare requester.

For APPC (LU6.2) communications with the CICS Clients or with other CICS systems:

  • IBM eNetwork Communications Server for OS/2 Warp, Version 5.0
  • IBM Communications Server for OS/2 Warp, Version 4.0, or later

For TCP/IP communications with the CICS Clients or with other CICS systems, IBM TCP/IP for OS/2 Warp Version 2.0, or later:

  • With OS/2 Warp Server Version 4, OS/2 Warp Version 4, and OS/2 Warp Connect Version 3, TCP/IP support is included within the operating system.

For PNA attachment of ASCII displays to CICS for OS/2 Version 3.1, PNA Version 1.2 is required.

For relational database functions:

  • IBM DB2® Universal Database, Version 5.0
  • IBM Database Server for OS/2 Warp, Version 4
For commercial messaging:
  • IBM MQSeries™ for OS/2 Warp, Version 5.0

Installation Instructions

To Install from the CD:

  1. Change to the \ts4os2\country directory on the CD, where country is a two-character representation of the country version you wish to install.
  2. Type install to start the installation.

Enrolling CICS for OS/2 Licenses and Users

  1. Select the License Use Runtime - Client folder from the CICS for OS/2 folder.
  2. Select the Nodelock Administration Tool.
  3. Select Products from the action bar, then New. The New product panel is displayed.
  4. Select Import. The Import panel displayed.
  5. Point to the LICENSE.LIC file located in the \ts4os2 directory and select OK.
  6. Important: You must select the Concurrent checkbox. When you have done this, enter the number of licenses in the Licenses scroll box, and select OK.
You have now enrolled your CICS for OS/2 license.

For more information, see Hints, Tips and Techniques for CICS for OS/2.
