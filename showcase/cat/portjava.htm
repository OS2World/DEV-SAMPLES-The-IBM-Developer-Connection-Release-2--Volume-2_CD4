Porting Windows and OS/2 based Applications to Java - R2V1 Dec. 1997 (M4) (N)

This product is located on Member Disc 4.

grid_layout.java is the sample code referenced in the article by Kelvin Lawrence that was published in the December 1997 issue of the IBM Developer Connection Technical Magazine. The article is entitled Porting Windows and OS/2 Based Applications to Java. The article is an introduction to using and exploiting Java's Abstract Window Toolkit (AWT).

This code is a simple Java program that creates a frame window and uses the GridLayout manager to create a three-by-three grid of pushbuttons as discussed in the article. The article is available on the Developer Connection Web site at www.developer.ibm.com/devcon/.

Installation Instructions

To Install from the CD:

  1. Copy the contents of \portjava on the CD to your system.
  2. Refer to the above article for more information.
