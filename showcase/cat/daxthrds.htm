DAXTHRDS - VisualAge C++ Sample (M4)

This product is located on Member Disc 4.

This is a sample showing how the various background thread enabled parts can be used in an application.

Hardware and Sofware Requirements:

  • IBM VisualAge C++
  • FixPak CTV304
  • 155 KB disk space

Web Site:

  • www.software.ibm.com/ad/visualage_c++

Installation Instructions:

  1. Unzip \daxthrds\daxthrds.zip into its own directory.
  2. Copy daxthrds.vbb to your \ibmcpp\dde4vb directory. You might wish to mark the daxthrds.vbb file as read only to prevent it from being updated accidentally.
  3. Copy idathrd.hpp to your \ibmcpp\include directory.
  4. Copy cppods3i.lib to your \ibmcpp\lib directory.
  5. Copy cppods3i.dll to your \ibmcpp\dll directory.
  6. Copy daxscl.bnd to your \ibmcpp\bnd directory.
  7. Execute sglbind x:\ibmcpp\bnd\daxscl.bnd <yourDatabase> to rebind the new version of the class library to any database that uses it.
  8. Relink any application using the DAB support and the cppods3i.dll.

    The DAXTHRDS package will co-exist with the non-thread enabled version.

  9. To build the sample, follow the instructions in the \daxthrds\readme.txt file.
