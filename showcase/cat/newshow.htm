NewShow: A C++ DIVE class sample-Volume 9 (M4)

This product is located on Member Disc 4.

NewShow is a sample written in C++ using the Direct Video Extensions (DIVE) of OS/2 Warp and the IBM Open Class libraries of VisualAge C++ 3.0.

This sample is functionally equivalent to the Show (jet) sample currently in the Toolkit (samples\mm\dive), and runs just as fast as its C counterpart. Hence, NewShow illustrates how to write an easy-to-use Dive class in C++, without sacrificing speed and performance. This sample accompanies the article "NewShow: A C++ Dive sample" found in The Developer Connection for OS/2 News, Volume 9.

Hardware and Software Requirements:

  • OS/2 Warp
  • IBM VisualAge C++ 3.0
  • Multimedia support must be installed for DIVE (installed automatically)

Installation Instructions:

To install from the CD:

  1. Create a \newshow directory on your hard drive.
  2. Copy the files in the X:\newshow directory to the \newshow directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to the \newshow\readme file for more information.
