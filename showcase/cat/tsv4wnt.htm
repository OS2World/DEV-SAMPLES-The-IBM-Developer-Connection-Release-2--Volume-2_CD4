Transaction Server for Windows NT V4.0 (P5)

This product is located on Premier Disc 5.

IBM Transaction Server for Windows NT, Version 4 is a member of the IBM Software Server series, a family of modular application servers. These servers enable you to rapidly implement client/server applications, and extend application capabilities to meet future business requirements.

Transaction Server for Windows NT, Version 4 features in one, integrated package:

  • CICS for Windows NT (Version 4.0)

  • Encina for Windows NT (Version 2.2)

  • CICS Clients (Version 2.0.1, for Windows NT, Windows 95, OS/2, DOS, Windows 3.1, and Macintosh)

  • CICS Client for AIX (Version 2.1.1)

  • Encina clients for Windows NT and AIX (Version 2.2)

All of these products at these new levels are delivered only in the Transaction Server for Windows NT, Version 4 and cannot be ordered separately.

The CICS Clients for Windows NT and Windows 95 provide an OLE automation server that enables the client function to be accessed from products that support OLE, including Microsoft Visual Basic, Microsoft Visual Basic for Applications, Lotus Script, Microsoft VBScript with ActiveX, and others. Samples are available in the CICS SupportPacs library that can be accessed on the Internet at: www.hursley.ibm.com/cics/txppacs/txpsumc.html.

The IBM CICS Gateway for Java for OS/2 and CICS Gateway for Java for Windows NT, Version 1.1, are included in the Transaction Server for Windows NT, Version 4 program package. The CICS Gateway for Java was made available, at Version 1.0, on the Internet on October 29, 1996. Announcement material describing the CICS Gateway for Java can be viewed, and the Gateway downloaded without charge, at the CICS home page at: www.hursley.ibm.com/cics/.

The Version 1.1 level of the CICS Gateway for Java was announced on December 20, 1996. Later levels may be available at the above URL. In addition to its support for the CICS Client family and for the CICS Client for AIX, CICS for Windows NT Version 4.0 also supports operation with the CICS for SINIX Client and the CICS for Solaris Client. These clients are delivered respectively in CICS for SINIX Version 2.1 and in CICS for Solaris Version 2.1.1.

Hardware and Software Requirements

  • An Intel machine with a minimum processor of a 486 running at 75 MHz (Memory and hard file requirements depend on the components to be run.)
  • Windows NT Version 3.51 or 4.0

    The Transaction Server includes the DCE Runtime Services prerequisite.

Installation Instructions:

To Install from the CD:

  1. Change to the \tsv4wnt\SERVERS\country directory on the CD, where country is a two-character code for the country version you wish to install.
  2. Type setup to start the installation.
  3. Refer to README.1ST in your country directory for more information.

Enrolling CICS for Windows NT Licenses and Users

  1. Select the License Use Runtime - Client folder from the CICS for Windows NT folder.
  2. Select the Nodelock Administration Tool.
  3. Select Products from the action bar, then New. The New product panel is displayed.
  4. Select Import. The Import panel displayed.
  5. Point to the LICENSE.LIC file located in the \tsv4wnt directory and select OK.
  6. Important: You must select the Concurrent checkbox. When you have done this, enter the number of licenses in the Licenses scroll box, and select OK.
You have now enrolled your CICS for Windows NT license.
