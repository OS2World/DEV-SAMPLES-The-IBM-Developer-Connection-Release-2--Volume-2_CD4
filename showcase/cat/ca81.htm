CICS Client for OS/2 CGI Samples (M4)

This product is located on Member Disc 4.

The sample COBOL programs supplied in this SupportPac show how a CICS application can be accessed from the World Wide Web using the External Call Interface (ECI) available as part of the CICS Client for OS/2.

The programs interact with a supplied CICS for OS/2 backend application which is part of the TECHDEMO developed by the Transaction Processing Marketing Team at IBM's Hursley Laboratories in the UK.

The sample programs are supplied as both source files and compiled executables. They illustrate how COBOL can be used to write a CGI program that will interface with the CICS External Call Interface and thus permit access to CICS server applications from the WWW.

Skill level required is familiarity with CICS for OS/2, COBOL and CGI programming.

Hardware and Software Requirements

The following products and versions are required:

  • A CICS for OS/2 Client. The MicroFocus COBOL sample requires the CICS for OS/2 Client V1.0 or later. The IBM COBOL sample requires the CICS Client for OS/2 V1.1 or later.
  • A WWW server. The IBM Internet Connection Server (ICS) V4.0, under which this application was developed and tested, is recommended. (A copy of ICS V4.0 can be obtained for free from the following URL: http://www.ics.raleigh.ibm.com
    The SupportPac installation instructions provided in the User Guide assume this server is being used and is installed in it's default directories.
  • A CICS for OS/2 V2.0.1 or Warp Transaction Server to run the CICS backend application.
