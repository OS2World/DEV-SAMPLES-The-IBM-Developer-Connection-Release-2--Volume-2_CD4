ISA Device Driver Samples (M4)

This product is located on Member Disc 4.

This package contains a number of subdirectories with sample code. The README explains what is in each subdirectory. The subdirectories and their short descriptions are listed below:

  • "ISA: Diskette Device Driver" /isa/fd subdirectory contains a sample floppy diskette adapter driver
  • "ISA: Keyboard Device Driver" /isa/isakbd subdirectory contains a sample keyboard driver
  • "ISA: Mouse Device Driver" /isa/isamouse subdirectory contains a sample mouse adapter driver
  • "ISA: IBM 16/4 PowerPC Token-ring driver" /isa/itok subdirectory contains a sample token ring adapter driver
  • "ISA: Parallel Printer Device Driver" /isa/ppa subdirectory contains a sample parallel port driver
  • "ISA: Sample Streams tty driver" /isa/tty subdirectory contains a sample serial port driver. This one contains a complete build environment for the sample to compile/link and run.

Installation Instructions

To Install from the CD:

  1. Copy the file isaddsamp.tar.Z from the
    \isaddsam directory on the cd, to a local directory (eg., \isaddsam).
  2. Uncompress the file by typing uncompress isaddsamp.tar.Z
  3. Untar the file by typing tar -xvf isaddsamp.tar
