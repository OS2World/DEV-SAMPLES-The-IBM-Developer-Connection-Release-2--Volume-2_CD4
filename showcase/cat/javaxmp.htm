Java Example Code (M4) (N)

This product is located on Member Disc 4.

This example code illustrates the principles of programming with the Java Abstract Window Toolkit and Java Foundation Classes. The code is used as a teaching aid to accompany seminars and presentations given by Kelvin Lawrence and Dan Becker. Both men have published articles in the IBM Developer Connection Technical Magazine and are employed by IBM in Austin, Texas.

Installation Instructions

To Install from the CD:

  1. Copy the contents of the \javaxmp directory on the CD, to a local temp directory.
  2. Recursively unzip the javaxmp.zip file in the temp directory.
  3. Review the \javaxmp\readme file on the CD for complete installation instructions.
