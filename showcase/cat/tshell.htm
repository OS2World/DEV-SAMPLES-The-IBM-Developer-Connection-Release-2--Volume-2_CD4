T Shell-Volume 3 (M4)

This product is located on Member Disc 4.

TShell is a text (non-GUI) shell for OS/2. TShell provides multiple OS/2 sessions and can start DOS and WIN-OS/2 sessions if the configuration is set to do this. OS/2 with TShell will not run OS/2 PM applications because the PMWIN window manager is not initialized.

Notes:

  1. Specify TSHELL.EXE in the PROTSHELL statement in CONFIG.SYS.
  2. Optional: Use supplied PGMSHELL.EXE to program the TShell start list.

Refer to "The Shell Game--Alternative Shells for OS/2" in The Developer Connection News, Volume 3, for a discussion of TShell.

Installation Instructions:

To install from the CD:

  1. Install a plain-text editor, such as TEDIT.EXE, on OS/2 Warp.
  2. Make a backup copy of CONFIG.SYS.
  3. Copy \tshell\tshell.exe from the CD into the root directory of the bootup drive.
  4. Edit CONFIG.SYS and change the PROTSHELL setting to PROTSHELL=\TSHELL.EXE
  5. Refer to the \tshell\readme file for more information.
