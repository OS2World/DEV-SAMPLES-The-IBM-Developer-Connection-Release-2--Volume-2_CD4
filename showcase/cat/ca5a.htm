CICS Client for Windows - Programming with LotusScript (M4)

This product is located on Member Disc 4.

This SupportPac provides a sample Lotus Notes Database containing two Forms showing how to use the CICS Client V2.0 OLE interfaces to access data from CICS using either the CICS External Call Interface (ECI) or the CICS External Presentation Interface (EPI).

Possible Uses

'ECIForm' uses the CICS External Call Interface (ECI) to call a CICS server program, 'EPIForm' uses the CICS External Presentation Interface (EPI) to run a BMS transaction on the CICS server. C source code for the server backend programs is provided.

Hardware and Sofware Requirements:

  • CICS Client v2.01 for Windows NT or Windows 95
  • Lotus Notes v4.0 or later for the appropriate platform
  • 258Kb hard disk space

Web Site:

  • www.hursley.ibm.com/cics

Installation Instructions:

To install from the CD:

  1. Copy \ca5a\ca5a.zip to a temporary directory.
  2. Uncompress the file using InfoZip's Unzip.
  3. Use the default options to ensure the subdirectories stored in the ZIP file are re-created.
  4. Follow the instructions in the readme file to complete the installation.
