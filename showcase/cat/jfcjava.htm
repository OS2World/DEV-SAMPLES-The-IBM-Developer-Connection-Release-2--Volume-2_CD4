THE JAVA FOUNDATION CLASSES: the NEW STANDARD for Java GUI Development - R2V2 June 1998 (M4) (N)

This product is located on Member Disc 4.

JFCSliders.java contains the sample code referenced in the article by Kelvin Lawrence that was published in the June 1998 issue of the IBM Developer Connection Technical Magazine. The article is entitled The Java Foundation Classes: The New Standard for Java GUI Development. The article discusses the Lightweight User Interface Component of the JFC and is available from the Developer Connection Web page as well as the technical magazine.

The sample code is used to show how small Java programs can be as a result of utilizing the power and flexibility that is built into the Java language itself and into the Java Foundation Classes.

This sample code is located in the \jfcjava directory.

There is artpag20.htm file in member disk 2 under \news0698\ directory. See this file for more information.
