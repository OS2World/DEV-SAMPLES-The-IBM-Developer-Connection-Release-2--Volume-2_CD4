OS2TUTOR-Volume 8 (M4)

This product is located on Member Disc 4.

OS2TUTOR provides frameworks that enable you to create tutorials that look and feel like the OS/2 Warp tutorial.

Hardware and Software Requirements:

  • OS/2 2.0 or later
  • IBM C Set Compiler
  • 1.5 MB DASD

Installation Instructions:

To install from the CD:

  1. Create an \os2tutor directory on your hard drive.
  2. Copy the files in the X:\os2tutor directory to the \os2tutor directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to The Developer Connection News - Volume 8 for more information.
