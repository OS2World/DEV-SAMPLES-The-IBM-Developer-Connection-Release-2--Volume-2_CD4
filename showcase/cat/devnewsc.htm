Exception Management in 32-Bit OS/2 - Volume 2 (M4)

This product is located on Member Disc 4.

This article features a discussion of exception management in 32-bit OS/2 and includes two Workplace Shell sample programs written in C.

The two sample programs show:

  • Sparse allocation, where pages in a memory allocation are committed only when referenced
  • Graceful failure, where functions in EXEs and DLLs return sensible, failing, result codes in trap situations

Installation Instructions:

To install from the CD:

  1. Create a \devnewsc directory on your hard drive.
  2. Copy the files in the X:\devnewsc directory to the \devnewsc directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to the \devnewsc\readme file for more information.
