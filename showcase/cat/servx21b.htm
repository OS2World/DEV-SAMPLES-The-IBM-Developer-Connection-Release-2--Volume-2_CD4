ServletExpress V2.1 (Beta) (M2)

This product is located on Member Disc 2.

Servlet Express is a servlet engine plug-in that turns ordinary Web servers into Java-enabled Web servers. Servlets allow for the creation of complicated, high-performance, cross-platform Web applications. Servlet engines are needed for loading and executing the servlets.

Servlet Express enables the use of the more advanced and stable JDK 1.1.1 even if a Web server already uses a Java VM. Servlet Express is superior to other servlets because it is an in-process plug-in and therefore provides better performance. In addition, installation is simple -- all you need is the JDK, 2 DLLs, and one zip file.

Hardware and Software Requirements

Platform: Windows NT
Web Server: ICS 4.1.1, ICS 4.2, MS IIS 2, MS IIS 3, Netscape FastTrack 2.01, Netscape Enterprise 2.01, Netscape Enterprise 3.0b3.
