I3STATE - VisualAge C++ Sample (M4)

This product is located on Member Disc 4.

This example shows how to use IObjectWindow and to create a part out of it. It shows how to receive messages off of the message queue and to process them, and if necessary, send out part notifications.

This package contains a sample implementation of a 3 state checkbox part for use with the Visual Builder in VisualAge C++ V3.0.

Hardware and Sofware Requirements:

  • IBM VisualAge C++
  • Visual Builder
  • 5 KB disk space

Web Site:

  • www.software.ibm.com/ad/visualage_c++

Installation Instructions:

  1. Unzip \i3state\i3state.zip to extract the sample code.
  2. Follow the instructions in the read.me file, which is included in i3state.zip.
