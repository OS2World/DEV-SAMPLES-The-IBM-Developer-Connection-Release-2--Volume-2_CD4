Monitoring Display Driver Interface Calls-Volume 6 (M4)

This product is located on Member Disc 4.

PMDDHOOKS is for software developers who want to learn how to monitor function calls going to the current display driver. This sample code is a simple illustration to show how to set up one hook module, and hooks out one function which you can modify.

Refer to "Monitoring Display Driver Interface Calls" in The Developer Connection News, Volume 6, for more information about this process.

Installation Instructions:

To install from the CD:

  1. Copy the X:\pmddhk directory to your hard drive. (X represents your CD-ROM drive.)
  2. Refer to the \pmddhk\readme.txt file and the above article for more information.
