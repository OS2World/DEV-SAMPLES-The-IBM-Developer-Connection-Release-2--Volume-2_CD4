Dynamic Linking on OS/2 V2.X - Volume 2 (M4)

This product is located on Member Disc 4.

This article demonstrates how easy it is to build and use 32-bit DLLs on OS/2 2.x, especially if you use the IBM C Set ++ compiler. A sample program is provided. It uses runtime dynamic linking and shows how to write a DLL entry-point function using the features of C Set ++.

Installation Instructions:

To install from the CD:

  1. Create a \devnewsb directory on your hard drive.
  2. Copy the X:\devnewsb directory to the \devnewsb directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to The Developer Connection News - Volume 2 for more information.
