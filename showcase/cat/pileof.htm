Pile of Class-Volume 8 (M4)

This product is located on Member Disc 4.

The PILEOF sample is a demonstration of the PileOf object class. This object will inherit most of the attributes of the WPFolder class, but it will override some methods to implement a given behavior.

A PileOf object should:

  • Enable the user to select what type of objects can be part of a given pile
  • Have a particular "open" behavior when the user double-clicks on a PileOf object icon
  • Support certain behavior for manipulating entries and for the creation of other piles

An instance of this class can be created as a Workplace object.

Note: This is sample code. Not all the PileOf behavior has been included.

Please take note that DosQueryModuleHandle ("pileof.dll -two instances- in \pileof\pileof.c") is failing because the path for pileof.dll is not hard coded.

In config.sys, make sure your LIBPATH statement points to the directory in which the pileof.dll was installed.

Installation Instructions:

  1. Change to the directory where the PileOf install.exe resides.
  2. At an OS/2 command prompt, type: install d:\pileof\pileof.dll

    (where d:\pileof  represents the drive letter and directory where pileof.dll is located on your machine.)

    This command creates the PileOf object on your desktop.

  3. Refer to the \pileof\readme.now file for more information.
