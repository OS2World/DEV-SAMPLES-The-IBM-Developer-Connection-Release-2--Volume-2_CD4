Memory Mapped File I/O for OS/2 - R2V1 Sept. 1997 (M4)

This product is located on Member Disc 4.

Code snippets and the tool to convert R/O data files into linkable object files as discussed in the article, Memory Mapped File I/O for OS/2: An Alternative Solution, by Christopher Matthews. This article was printed in the June 1997 issue of the IBM Developer Connection News magazine.

The complete article is available from the Developer Connection Web site at service.software.ibm.com/devcon/news/r2v1/7.htm.

Installation Instructions

To Install from the CD:

  1. Recursively copy the contents of \memmapio to your system.
  2. Refer to the above article for more information.
