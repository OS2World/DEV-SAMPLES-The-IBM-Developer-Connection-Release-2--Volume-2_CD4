STRGEN - VisualAge C++ Sample (M4)

This product is located on Member Disc 4.

This is a sample of using the IStringGenerator class for translating elements to IStrings in the ICollectionViewListBox. This sample can also be used if you wish to populate table records from DAX classes to the ICollectionViewListBox.

Hardware and Sofware Requirements:

  • IBM VisualAge C++
  • 1.2 MB disk space

Web Site:

  • www.software.ibm.com/ad/visualage_c++

Installation Instructions:

  1. Unzip \strgen\strgen.zip to extract the sample code and executable.
  2. Follow the instructions in the readme file, which is included in strgen.zip.
