Writing Object Oriented Multimedia IO Procs-Volume 4 (M4)

This product is located on Member Disc 4.

This is the sample source code for writing an IOProc Base Class in C++. The following files are included:

  • MYIOPROC.CPP Source file
  • IOPROC.HPP IOProc base class header file
  • IOPROC.CPP IOProc base class code file
  • MAKEFILE Source file

Refer to "Writing Object Oriented Multimedia IO Procedures" in The Developer Connection News, Volume 4.

Installation Instructions:

To install from the CD:

  1. Create a \objmmio directory on your hard drive.
  2. Copy the files in the X:\objmmio directory to the \objmmio directory on your hard drive. (X represents your CD-ROM drive.)
