Multithreading PM Applications - Volume 1 (M4)

This product is located on Member Disc 4.

This article discusses a two-threaded PM application architecture that uses the "start-a-thread, keep-it-around, and give-it-work" approach. It includes a sample program written in C.

Installation Instructions:

To install from the CD:

  1. Create a \devnewsa directory on your hard drive.
  2. Copy the files in the X:\devnewsa directory to the \devnewsa directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to the \devnewsa\readme file for more information.
