CICS Client for OS/2 - OS/2 PM ECI Samples (M4)

This product is located on Member Disc 4.

This SupportPac is one of a series providing sample C programs that illustrate the usage of the External Call Interface (ECI) feature available with the IBM CICS Client family of products.

The samples provided with this SupportPac are OS/2 PM programs to be used in conjunction with the IBM CICS Client for OS/2 product. The C source for the back end application used by the samples is also provided for compiling on the target CICS Server together with compiled versions for CICS for OS/2 V2 (16-bit) and Transaction Server for OS/2 (32-bit).

Hardware and Sofware Requirements:

  • CICS client for OS/2
  • 1.1 MB disk space

Web Site:

  • www.hursley.ibm.com/cics

Installation Instructions:

To install from the CD:

  1. Copy \ca51\ca51.zip to a temporary directory.
  2. Uncompress the file using InfoZip's Unzip.
  3. Use the default options to ensure the subdirectories stored in the ZIP file are re-created.
  4. Follow the instructions in the readme file to use the samples.
