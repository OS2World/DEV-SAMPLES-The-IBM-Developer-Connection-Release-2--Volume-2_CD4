CNRHELPW - VisualAge C++ Sample (M4)

This product is located on Member Disc 4.

This is a Visual Builder sample that shows how to provide context-sensitive help for containers and the container itself.

This sample is for Windows only. See the CNRHELPO sample for the OS/2 version.

Hardware and Sofware Requirements:

  • IBM VisualAge C++ for Windows
  • Visual Builder
  • 134 KB disk space

Web Site:

  • www.software.ibm.com/ad/visualage_c++

Installation Instructions:

    Unzip \cnrhelpw\cnrhelpw.zip to extract the sample code and executable.
