Using the Developer API Extensions (Open32)-Volume 10 (M4)

This product is located on Member Disc 4.

This sample accompanies an article in The Developer Connection News, Volume 10 demonstrating all of the major steps of porting Windows applications to OS/2 using Open32.

Open32 was previously known as the IBM Developer API Extensions, or DAPIE.

Installation Instructions:

To install from the CD:

  1. Copy the X:\dapiesrc directory to your hard drive. (X represents your CD-ROM drive.)
  2. Refer to the above article for more information.
