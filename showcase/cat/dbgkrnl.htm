Installation Instructions:

To install from the CD:

  1. Create a \dbgkrnl directory on your hard drive.
  2. Copy the files in the X:\dbgkrnl directory to the \dbgkrnl directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to the \dbgkrnl\new_dbkl.dsc file for more information.
