StartDos-Volume 3 (M4)

This product is located on Member Disc 4.

StartDos is a utility program used to start DOS sessions programmatically. It is useful for running programs under reduced-function shells like MShell or TShell, or when spawning DOS programs from OS/2 batch files.

StartDos uses REXX to specify DOS settings before the session starts.

Refer to "The Shell Game--Alternative Shells for OS/2" in The Developer Connection News, Volume 3, for a discussion of alternative shells.

Installation Instructions:

To install from the CD:

  1. Create a \startdos directory on your hard drive.
  2. Copy the files in the X:\startdos directory to the \startdos directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to the \startdos\readme file for more information.
