OS/2-Win-OS/2 Communications - Volume 2 (M4)

This product is located on Member Disc 4.

The sample code in the article "Communication between OS/2
and Windows Processes" in Volume 2 of The Developer Connection News contains some errors.

This product provides the following corrected code samples:

  • OpenVirtQueue
  • WriteVirtQueue
  • ReadVirtQueue

Some of the samples will be used later for fully functional applications to communicate between OS/2 and Windows.

Installation Instructions:

To install from the CD:

  1. Create a \devnewsd directory on your hard drive.
  2. Copy the files in the X:\devnewsd directory to the \devnewsd directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to The Developer Connection News - Volume 2 for more information.
