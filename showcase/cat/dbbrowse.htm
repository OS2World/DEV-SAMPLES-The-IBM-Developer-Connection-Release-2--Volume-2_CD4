DBBROWSE - VisualAge C++ Sample (M4)

This product is located on Member Disc 4.

This utility uses some classes under development in the Data Access Builder team to build applications within the Visual Builder for dynamic access to data sources. It can be used to browse the available datasources, as well as connect to, then browse the contents of the datasources. This utility can be used to help resolve connection problems when using the DAB and generated code - errors should be reported as error messages in DBBrowse, which can help with problem determination.
