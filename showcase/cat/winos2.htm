OS/2-Win-OS/2 Communications-Volume 4 (M4)

This product is located on Member Disc 4.

This product provides sample code for the article "Communication between OS/2 and WIN-OS/2" in Volume 4 of The Developer Connection News.

The article provides a working model that allows the OS/2 and WIN-OS/2 process to send messages to each other, as well as display them to the console. You can expand on this simple model so the messages can be broadcast to all processes that are registered, or to a designated process within a group of registered processes.

Installation Instructions:

To install from the CD:

  1. Create a \winos2 directory on your hard drive.
  2. Copy the files in the X:\winos2 directory to the \winos2 directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to The Developer Connection News - Volume 4 for more information.
