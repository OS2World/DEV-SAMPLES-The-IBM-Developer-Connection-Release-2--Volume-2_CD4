Hello World: Part II, OpenDoc Views and Presentation Types-Volume 7 (M4)

This product is located on Member Disc 4.

This source code is related to the article "Part II, OpenDoc Views and Presentation Types" in The Developer Connection News, Volume 7. Refer to this article for a discussion of the OpenDoc view and presentation types. This is the second installment in a series on OpenDoc sample part code.

Hello World is a simple example of how to subclass existing sample code in the OpenDoc toolkit.

Installation Instructions:

To install from the CD:

  1. Create a \hello2 directory on your hard drive.
  2. Copy the files in the X:\hello2 directory to the \hello2 directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to the \hello2\readme.txt file for more information.
