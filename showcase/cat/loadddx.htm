Loadddx for GXT110P 2M Video Card (For S3 Trio64V+ graphics chip) (M4)


This is unoptimzed example code. The code will produce a loadddx for the GXT110P 2M video card. In order to keep the complexity down, the code does not take advantage of the graphics hardware.

Note: This code is designed for the S3 Trio64V+ graphics chip.
