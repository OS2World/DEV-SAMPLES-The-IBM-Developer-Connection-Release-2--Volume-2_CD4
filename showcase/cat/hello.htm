Writing OpenDoc Part Handlers: The Hello World Part-Volume 6 (M4)

This product is located on Member Disc 4.

OpenDoc provides a set of services that enable developers to write modules that they can integrate into a document containing multiple data types. These modules are known as "part handlers."

Hello World is a simple example of how to subclass existing sample code in the OpenDoc toolkit.

Installation Instructions:

To install from the CD:

  1. Copy the X:\hello directory to your hard drive. (X represents your your CD-ROM drive.)
  2. Refer to the article, "Writing OpenDoc Part Handlers: Hello World" in The Developer Connection News, Volume 6, for a description of "Hello World."
