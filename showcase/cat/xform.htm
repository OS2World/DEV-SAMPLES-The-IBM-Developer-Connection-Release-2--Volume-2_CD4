Transform Example-Volume 3 (M4)

This product is located on Member Disc 4.

Transform Example contains samples to go with the article by Kelvin Lawrence in Volume 3 Developer Connection News. The source was compiled using the IBM C-Set++ compiler.

Installation Instructions:

To install from the CD:

  1. Create an \xform directory on your hard drive.
  2. Copy the files in the X:\xform directory to the \xform directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to The Developer Connection News - Volume 3 for more information.
