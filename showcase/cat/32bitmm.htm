32-bit Memory Management-Volume 10 (M4)

This product is located on Member Disc 4.

This article discusses programming the 32-bit memory model of OS/2. It covers memory addressing, allocating, and heap management. It also covers memory leaks and how to debug them. Sample code is provided in C.

Installation Instructions:

To install from the CD:

  1. Copy the X:\32bitmm directory to your hard drive. (X represents your CD-ROM drive.)
  2. Refer to the \32bitmm\readme file and The Developer Connection News - Volume 10 for more information.
