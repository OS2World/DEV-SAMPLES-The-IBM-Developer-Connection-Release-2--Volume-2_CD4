M Shell-Volume 3 (M4)

This product is located on Member Disc 4.

MShell is a mini PM shell for OS/2 2.x. It is a program launcher that uses the RUNWORKPLACE architecture of the Workplace Shell. You can configure programs by editing a plain-text .INI file.

Refer to "The Shell Game - Alternative Shells for OS/2" in The Developer Connection News, Volume 3, for a discussion of MShell.

Installation Instructions:

To install from the CD:

  1. Copy \mshell\mshell.exe from the CD into the root directory of the boot drive.
  2. Edit CONFIG.SYS and modify the SET for RUNWORKPLACE to specify mshell.exe. For example: SET RUNWORKPLACE=C:\MSHELL.EXE
  3. If you boot OS/2 from a drive other than C:, modify RUNWORKPLACE accordingly. When mshell.exe runs, it looks for mshell.ini in the root directory of the boot drive. If there is no mshell.ini, mshell will create one for you.
  4. Shut down and reboot.
