Making Your OS/2 Device Driver APM-Aware-Volume 5 (M4)

This product is located on Member Disc 4.

This sample code provides an example of how to establish the communication between a client device driver and the APM (Advanced Power Management) device driver for the purpose of power management event notification.

Installation Instructions:

To install from the CD:

  1. Copy the X:\fschroed directory on the CD to your hard drive. (X represents your CD-ROM drive.)
  2. Refer to The Developer Connection News - Volume 5 for more information.
