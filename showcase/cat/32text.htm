32-bit Char_Mode APIs-Volume 3 (M4)

This product is located on Member Disc 4.

OS/2 Warp Connect (PowerPC Edition) provides 32-bit interfaces to the text mode functions (Vio, Kbd, Mou). This library provides matching interfaces which can be run on OS/2 2.x and later and OS/2 Warp.

These interfaces provide for a common interface to the text mode of OS/2 on both the Intel and PowerPC platforms. These interfaces do not provide functions not in OS/2 2.x and later or OS/2 Warp, but rather they give a 32-bit interface to the existing function.

Installation Instructions:

To install from the CD:

  1. Create a \32text directory on your hard drive.
  2. Copy the files in the X:\32text directory to the \32text directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to the \32text\read.32t file for more information.
