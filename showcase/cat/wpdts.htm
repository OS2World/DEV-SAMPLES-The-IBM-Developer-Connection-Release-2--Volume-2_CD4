Writing Workplace Shell objects using IBM Open Class Libraries-Volume 9 (M4)

This product is located on Member Disc 4.

This sample accompanies an article in the Developer Connection News concerning how to write Workplace Shell objects using IBM Open Class.

For the non-DTS sample, IBM Open Class or IBM Class Library: User Interface (ICL:UI) is required. For the DTS sample, IBM Visual Age C++ is required.

Installation Instructions:

To install from the CD:

  1. Create a \wpdts directory on your hard drive.
  2. Copy the files in the X:\wpdts directory to the \wpdts directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to The Developer Connection News - Volume 9 for more information.
