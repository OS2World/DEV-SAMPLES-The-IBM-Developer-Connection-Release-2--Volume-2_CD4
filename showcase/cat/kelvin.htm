Fonts R Us-Volume 5 (M4)

This product is located on Member Disc 4.

"FONTS R US!" discusses some of the features that OS/2 Presentation Manager provides for manipulating and displaying text characters in different fonts.

Complete sample programs are provided on the Developer Connection CD.

Installation Instructions:

To install from the CD:

  1. Copy the X:\kelvin directory on the CD to your hard drive. (X represents your CD-ROM drive.)
  2. Refer to The Developer Connection News - Volume 5 for more information.
