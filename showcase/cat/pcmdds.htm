PCMCIA Device Driver Samples (M4)


/Samples/DDK411samples contains a number of subdirectories with sample code. The README explains what is in each subdirectory. The subdirectory and its short description is listed below:

  • "PCMCIA: Ethernet"
    /pcmcia/ent subdirectory contains a sample pcmcia ethernet adapter driver.

Installation Instructions:

  1. Copy /pcmdds/pcmddsamp.tar.Z to a temporary directory.
  2. Make the temporary directory the current directory.
  3. Type uncompress pcmddsamp.tar.Z
  4. Type tar -xvf pcmddsamp.tar
  5. See the /pcmdds/README.pcmddsamp file for more information.
