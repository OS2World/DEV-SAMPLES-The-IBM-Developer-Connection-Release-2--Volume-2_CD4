Bamba for Java Applet V0.98 Windows 95/NT (M4) (N)

This product is located on Member Disc 4.

Bamba for Java is a pure Java implementation of Bamba for streaming audio and video across the Web. No installation of plug-ins is required; this cross-platform solution will run on most computers with a browser running a Java Virtual Machine. The Bamba for Java applet is referenced by the HTML page, and the browser automatically downloads it from the server. The applet then fetches the Bamba audio or video clip and plays it on your computer. Bamba for Java plays the same *.iba clip s that Bamba plays.

Hardware and Software Requirements

Computer:
  • 486/DX2 *Video cannot be supported here.
  • Pentium 100Mhz *Video is supported depending on the frame rate and bit rate of the video clip.
  • Pentium 166Mhz *Audio with video up to 110 Kbps is supported.
Browser
  • v3.0 browser (Netscape or IE) supported, but 4.0 or higher is recommended.
Software
  • Any HTTPD Web Server software.
