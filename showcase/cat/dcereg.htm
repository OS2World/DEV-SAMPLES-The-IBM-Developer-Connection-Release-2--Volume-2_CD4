Mustering DCE Registry Objects with the Managed Object Class Library-Volume 10 (M4)

This product is located on Member Disc 4.

This is an accompanying example program to the "Mustering DCE Registry Objects with the Managed Object Class Library" article. This example assumes that the user is logged into DCE with an account that has permission to create accounts. The program runs against your local cell.

Installation Instructions:

To install from the CD:

  1. Copy the X:\dcereg directory to your hard drive. (X represents your CD-ROM drive.)
  2. Refer to The Developer Connection News - Volume 10 for more information.
