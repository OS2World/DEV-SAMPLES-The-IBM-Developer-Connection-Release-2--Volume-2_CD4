Sharing The Parallel Port-Volume 4 (M4)

This product is located on Member Disc 4.

These samples include all the source code described in "Sharing the Parallel Port" in The Developer Connection News, Volume 4. The article explains the calls required to request and release exclusive access to the parallel port. The article also describes the OS/2 2.1 level 3 device driver model as it relates to the initialization complete strategy command.

Installation Instructions:

To install from the CD:

  1. Create a \sharport directory on your hard drive.
  2. Copy the files in the X:\sharport directory to the \sharport directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to The Developer Connection News - Volume 4 for more information.
