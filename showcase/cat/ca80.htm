CICS for MVS/ESA EXCI CGI Sample Program (M4)

This product is located on Member Disc 4.

This SupportPac provides a sample program that demonstrates how the Internet Connection Server for MVS/ESA can be used to directly invoke a CICS for MVS/ESA program via the CICS for MVS/ESA External CICS Interface (EXCI).

The sample program provided by this SupportPac can be used as a framework to implement access from the World Wide Web to any CICS program able to work with a Distributed Program Link Commarea.

To Build the sample program the user will need skills to be able to compile and manipulate files within the OpenEdition Hierarchical File System (HFS). To allow the sample program to be driven from a Web Browser the user will need skills to be able to setup and configure the Internet Connection Server for MVS/ESA.
