DLG2CPP - VisualAge C++ Sample (M4)

This product is located on Member Disc 4.

This package contains a tool for converting OS/2 PM dialog source into compilable C++ ICLUI code that uses a multi-cell canvas.

Hardware and Sofware Requirements:

  • IBM VisualAge C++
  • 1 MB disk space

Web Site:

  • www.software.ibm.com/ad/visualage_c++

Installation Instructions:

  1. Unzip \dlg2cpp\dlg2cpp.zip to a temporary directory.
  2. Follow the instructions in the dlg2cpp.ins file, which is included in dlg2cpp.zip.
