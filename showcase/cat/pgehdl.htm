PGEHDL - VisualAge C++ Sample (M4)

This product is located on Member Disc 4.

This package contains a sample of a wrapper non-visual part for IPageHandler, which is used to handle the page selected event when a notebook page is selected.

Hardware and Sofware Requirements:

  • IBM VisualAge C++
  • 749 KB disk space

Web Site:

  • www.software.ibm.com/ad/visualage_c++

Installation Instructions:

  1. Unzip \pgehdl\pgehdl.zip to extract the sample code and executables.
  2. For more information, refer to the readme file, which is included in pgehdl.zip.
