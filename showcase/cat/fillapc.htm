Filling You In: Areas, Clippings & Paths-Volume 4 (M4)

This product is located on Member Disc 4.

This source code demonstrates the use of areas, paths, and clippings in Presentation Manager graphics applications.

Refer to "Filling You In: Areas, Paths, and Clippings" in The Developer Connection News, Volume 4, for a description of this source code.

Installation Instructions:

To install from the CD:

  1. Create a \fillapc directory on your hard drive.
  2. Copy the files in the X:\fillapc directory to the \fillapc directory on your hard drive. (X represents your CD-ROM drive.)
  3. Refer to The Developer Connection News - Volume 4 for more information.
