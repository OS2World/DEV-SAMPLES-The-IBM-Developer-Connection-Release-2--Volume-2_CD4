CICS Bitmap and Icon Files (M4)

This product is located on Member Disc 4.

This SupportPac supplies bitmaps and icons of the CICS logo for use on the OS/2 desktop.

Hardware and Sofware Requirements:

  • CICS client for OS/2
  • 1 MB disk space

Web Site:

  • www.hursley.ibm.com/cics

Installation Instructions:

To install from the CD:

  1. Copy \ca02\ca02.zip to a temporary directory.
  2. Uncompress the file using InfoZip's Unzip.

    This will create the .ico and .bmp files.
