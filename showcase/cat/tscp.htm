Transaction Server for Windows NT Client Pack (P5)

This product is located on Premier Disc 5.

The IBM CICS Clients family comprises workstation products that bring the advantages of client/server operation to your transaction processing. CICS Clients allow users to access transaction and programs on the entire family of CICS application servers.

If you want to leverage the value of your existing applications, CICS Clients allow you to access your existing host CICS applications immediately, with no programming effort, using 3270 emulation. Later, you can add graphical user interfaces to these applications using the CICS Client External Presentation Interface (EPI). Or you can bridge to other systems using the CICS gateways for Lotus Notes and the Internet.

The IBM CICS Clients family comprises:

  • IBM CICS Client for AIX
  • IBM CICS Client for DOS
  • IBM CICS Client for OS/2
  • IBM CICS Client for Windows
  • IBM CICS Client for Windows 95
  • IBM CICS Client for Windows NT
  • IBM CICS Client for Macintosh

The CICS Clients can communicate with the following CICS server products:  

  • CICS for OS/2 Version 2.0.1, and higher

  • Transaction Server for OS/2 Warp Version 4.0, which includes CICS for OS/2 Version 3.0

  • CICS for Windows NT Version 2

  • Transaction Server for Windows NT Version 4.0

  • 'CICS on Open Systems', that is:
    • CICS for AIX
    • CICS for HP 9000
    • CICS for Digital UNIX
    • CICS for Siemens Nixdorf SINIX
    • CICS for Sun Solaris.

  • Transaction Server for AIX Version 4.0

  • CICS/400 Version 3 Release 1, and higher

  • CICS for MVS/ESA Version 3 Release 3, and higher

  • CICS/MVS Version 2 Release 1.2

  • CICS/VSE Version 2 Release 2 and higher

Hardware and Software Requirements

Hardware requirements:

  • Any Intel computer that is capable of running Windows NT. Minimum 64 Mbyte memory and up to 350 Mbyte hard disk are required.
Software requirements:
  • Microsoft Windows NT Workstation V3.51 or later, or Microsoft Windows NT Server V3.51 or later
Language support:
  • Any of:
    • IBM Visual C++ for Windows, V3.5
    • Microsoft Visual C++ for Windows NT, V4.0 and later
    • Micro Focus Object COBOL for Windows NT, V4

Installation Instructions

To Install from a CD:

  • All of the clients are located in the \tscp\CLIENTS directory on the CD.
  • To install the AIX client, change to the AIX_CLNT subdirectory, and run SMIT.
  • To install any other client, change to the I86_CLNT subdirectory, and review the README.xxx file, where xxx is a three-letter designation of the platform on which the client will be installed.
