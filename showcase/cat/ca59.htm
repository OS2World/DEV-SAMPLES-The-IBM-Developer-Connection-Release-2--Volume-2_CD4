CICS Client for Windows - Programming with VBScript (M4)

This product is located on Member Disc 4.

This SupportPac shows how to access the CICS Client Object Oriented wrappers from VBScript with Microsoft Internet Explorer.

The SupportPac contains a sample HTML page that will list the servers defined to a machine with the CICS Client installed. From the list it is possible to select a server and query the status of the connection.

The HTML page contains Microsoft VBScript and requires Microsoft Internet Explorer to view the HTML page.

Hardware and Sofware Requirements:

  • CICS Client v2.0 for Windows NT or Windows 95
  • CICS Clients OO Beta (available from the CICS Beta Program at www.hursley.ibm.com/cics).
  • Microsoft Internet Explorer v3.0 or later
  • 14 KB disk space

Web Site:

  • www.hursley.ibm.com/cics

Installation Instructions:

To install from the CD:

  1. Copy \ca59\ca59.zip to a temporary directory.
  2. Uncompress the file using InfoZip's Unzip.
  3. Use the default options to ensure the subdirectories stored in the ZIP file are re-created.
  4. Follow the instructions in the readme file to complete the installation.
