CICS Client for Windows - COBOL ECI Samples (M4)

This product is located on Member Disc 4.

This SupportPac provides sample Cobol programs that illustrate the usage of the External Call Interface (ECI) available with the IBM CICS Client family of products.

The samples provided can be used in conjunction with the IBM CICS Client for Windows product. The COBOL source for the back end application used by the samples is also provided for compiling on the target CICS Server. Compiled versions are supplied for use on both CICS for OS/2 V2.X and Transaction Server for OS/2. COBOL runtime libraries are provided for both the Windows Client and Transaction Server.

Hardware and Sofware Requirements:

  • CICS client for Windows
  • 1.7 MB disk space

Web Site:

  • www.hursley.ibm.com/cics

Installation Instructions:

To install from the CD:

  1. Copy \ca55\ca55.zip to a temporary directory.
  2. Uncompress the file using InfoZip's Unzip.
  3. Use the default options to ensure the subdirectories stored in the ZIP file are re-created. Folow the instructions in the readme file to use the samples.
